ImageManipulator Utility for manipulations of images. All public methods in this class accept ONLY Bitmap objects and all functions will automatically set PixelSnapping.ALWAYS and smoothing to true for methods that return a Bitmap object.

crop

static function crop(_x:Number, _y:Number, _width:Number, _height:Number, _stage:Stage, bitmap:Bitmap = null) : Bitmap

Crops a bitmap and returns the cropped bitmap. If no bitmap is provided, a bitmap the dimensions of the crop area will be returned.

Parameters
_x :X value to begin crop
_y :Y value to begin crop
_width :Width value of the crop
_height:Height value of the crop
_stage :The stage that the DisplayObject to be cropped is on
bitmap :The Bitmap to be cropped
Returns
  • croppedBitmap :the cropped bitmap instance
